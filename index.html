What is Once Bytten?

A browser-native, zero-dependency byte-level file security scanner. Everything runs locally in your browser. No file ever leaves your device.

Threat Levels

  • CLEAN — No suspicious indicators found
  • LOW — Minor anomalies or informational flags
  • MEDIUM — Suspicious patterns, type mismatches, or embedded executables in archives
  • HIGH — Multiple high-risk indicators confirmed
  • CRITICAL — Confirmed malicious patterns or extreme structural anomalies

Tabs

  • Overview — Summary, threat badge, key findings
  • Magic Bytes — File type detection from raw bytes, extension mismatch
  • Entropy — Shannon entropy heatmap, byte frequency distribution
  • Strings — Extracted printable strings: URLs, IPs, paths, keywords, base64
  • Structure — Format-specific parsing: PE headers, ELF, ZIP contents, PDF flags
  • Hex View — Paginated raw hex viewer with search
  • ROSA — Suffix automaton structural analysis, bigram distribution map

Entropy Guide

  • 0–2: Very structured data (mostly zeros, ASCII text)
  • 2–5: Normal program code or text
  • 5–7: Compressed data (zip, gzip contents)
  • 7–8: Encrypted or packed data — common in malware payloads

ROSA Analysis

ROSA (Rapid Online Suffix Automaton) builds a minimal deterministic finite automaton over sampled file bytes. The state count and transition density reveal structural properties invisible to Shannon entropy — particularly useful for detecting hidden patterns in seemingly random data.

Xinu Compliance

Once Bytten is fully browser-native: zero external dependencies, zero network calls, single HTML file. The browser is the runtime.
